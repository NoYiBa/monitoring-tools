How to use it
=============


Settings
~~~~~~~~

This is the list of settings which can be redefined in the host definition

_GRAPHITEURL
------------

:type:              string
:description:       The Graphite path. Default: /

_GRAPHITEWARN
-------------

:type:              Integer
:description:       Warning threshold. Default: 3

_GRAPHITECRIT
-------------

:type:              Integer
:description:       Critical threshold. Default: 5
