How to use it
=============


Settings
~~~~~~~~

This is the list of settings which can be redefined in the host definition

_ESX_CPU_WARN
--------------

:type:              percent
:description:       Level for cpu usage


_ESX_CPU_CRIT
--------------

:type:              percent
:description:       Level for cpu usage


_ESX_MEM_WARN
--------------

:type:              percent
:description:       Level for ram usage


_ESX_MEM_CRIT
--------------

:type:              percent
:description:       Level for ram usage


_ESX_SWAP_WARN
--------------

:type:              percent
:description:       Level for swap usage


_ESX_SWAP_CRIT
--------------

:type:              percent
:description:       Level for swap usage
