How to use it
=============


Settings
~~~~~~~~

_DNSHOSTNAME
------------

:type:          string
:description:   Hostname to resolve

_DNSEXPECTEDRESULT
------------------

:type:          string
:description:   Address expected returned by the DNS server

_DNSWARN
--------

:type:          Integer
:description:   Warning threshold

_DNSCRIT
--------

:type:          Integer
:description:   Critical threshold

Triggers
~~~~~~~~

No trigger used with this pack
